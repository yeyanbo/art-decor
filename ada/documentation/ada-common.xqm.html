        getUri
        • view code
        • ( $prefix as xs:string ,
            $type as xs:string?) as xs:string

        Get the URI for a a subcollection in ADA. Function does not verfiy whether the collection exists,which would be just overhead if the caller is tested.

        • Parameters:
          • $prefix - Project prefix with or without trailing hyphen, i.e. 'demo1' or 'demo1-'
          • $type - Desired subcollection, i.e. 'data' or 'definitions', i.e.: 'definitions' or 'data'
        • Return:
          • URI string, i.e. '/db/apps/ada-data/projects/demo1/data/'
